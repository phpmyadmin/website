Translations

Translations of phpMyAdmin are handled using Gettext. It is used for both translating phpMyAdmin as well as the documentation (where Sphinx is being used).

You are welcome to contribute to any translations. You can follow instructions on the translators page. Or simply go to the translation server and translate there in the project phpMyAdmin. The translation server also provides you with an overview of the translation status.
